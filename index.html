About this tool

This is a self-assessment tool based on the A.G.E.N.T. framework, a five-question pre-deployment checklist for agentic AI.

Scoring

  • Each answer carries a risk weight from 0 (low) to 4 (critical).
  • Per-dimension score is the average of its five answers.
  • Any answer marked as a tripwire forces a BLOCK verdict on its own.

Verdicts

  • DEPLOY — overall < 1.5 and every dimension < 2.0.
  • GATE — overall < 2.5 and every dimension < 3.0.
  • BLOCK — any tripwire triggered, overall ≥ 2.5, or any dimension ≥ 3.0.

Security posture

  • Zero third-party JavaScript. Every script is served from this origin and auditable.
  • Strict Content-Security-Policy blocks any cross-origin fetch, form post, or frame.
  • No cookies, no analytics, no network calls. Data sits in localStorage only.
  • Reports include a SHA-256 hash of the answer set for tamper-evidence.

Tracking multiple agents

The tool assesses one agent at a time. For an organizational inventory, export a CSV after each assessment and track them in your GRC system of record. You can re-import any exported CSV later to review or update an assessment.
